# aud_etiology_classifier


Script which classifies unknown etiologies in audiology datasets using Naive Bayes models.


Model trained on known etiologies (with feature importance calculation), then implemented on data from unknown etiologies.

This script uses the open source Audiology (Standardized) Data Set available from:
https://archive.ics.uci.edu/ml/datasets/Audiology+%28Standardized%29
